# CatNFT - Ethereum NFT Smart Contract

This project is a simple NFT smart contract deployed on the Ethereum blockchain that uses a picture of a cat. The NFT adheres to the ERC721 standard and stores the image of the cat on IPFS.

## Getting Started

### Prerequisites

- Node.js and npm
- Truffle (`npm install -g truffle`)
- MetaMask wallet
- Ganache or Infura for deployment
